1. [Your First Year in Code PDF](https://leanpub.com/firstyearincode) (2019)
2. [Front-end Developer Handbook PDF](https://github.com/FrontendMasters/front-end-handbook-2019/raw/master/exports/Front-end%20Developer%20Handbook%202019.pdf) (2019)
3. [Principles of Programming Languages PDF](https://www.cs.swarthmore.edu/~zpalmer/cs73/s18/book.pdf) (2018)
4. [Learn Programming – Your Guided Tour Through the Programming Jungle](https://archive.org/details/2018LearnProgramming) (2018)
5. [Learning to Program: By Example PDF](https://leanpub.com/learning-to-program-by-example) (2016)
6. [What to Look for in a Code Review – Effective tips for reviewing code PDF](https://leanpub.com/whattolookforinacodereview) (2016)
7. [Learn Enough Command Line to Be Dangerous](https://www.learnenough.com/command-line-tutorial) (2016)
8. [Learn Enough Git to Be Dangerous](https://www.learnenough.com/git-tutorial/getting_started) (2016)
9. [97 Things Every Programmer Should Know](https://97-things-every-x-should-know.gitbooks.io/97-things-every-programmer-should-know/content/en/) (2015)
